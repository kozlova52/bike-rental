# Прокат велосипедов в Москве

## О проекте
Веб-приложение для аренды велосипедов в трёх парках Москвы.

## Функционал
- Регистрация и вход
- Просмотр велосипедов (BMX, Stels, Cube)
- Просмотр пунктов проката (Центральный парк, Парк Горького, ВДНХ)
- Оформление, завершение и удаление аренды
- Админ-панель

## Запуск проекта

### Бэкенд (NestJS)
```
cd backend
npm install
npm run start:dev
```


### Фронтенд (React)
```
cd frontend
npm install
npm run dev
```


### База данных (PostgreSQL)
- Создать БД `bike_rental`
- Восстановить из файла `DB/bike_rental_backup.sql`
